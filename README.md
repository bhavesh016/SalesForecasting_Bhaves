# Sales Forecasting and Demand Segmentation

## Project Overview

This project analyzes historical retail sales data and builds forecasting models to predict future sales. It also performs anomaly detection and product demand segmentation to support inventory and supply chain decisions. An interactive Streamlit dashboard is included for visualization and analysis.

---

## Dataset

- Superstore Sales Dataset
- File: `train.csv`

---

## Tasks Completed

### Task 1 – Exploratory Data Analysis (EDA)
- Data Cleaning
- Missing Value Analysis
- Monthly Sales Trend
- Category-wise Sales
- Regional Sales Analysis
- Shipping Analysis

### Task 2 – SARIMA Forecasting
- Monthly Sales Forecast
- Model Evaluation (MAE, RMSE, MAPE)

### Task 3 – Facebook Prophet Forecasting
- Future Sales Prediction
- Trend & Seasonality Analysis
- Model Evaluation

### Task 4 – XGBoost Forecasting
- Lag Feature Engineering
- Sales Prediction
- Model Evaluation

### Task 5 – Anomaly Detection
- Isolation Forest
- Monthly Sales Outlier Detection

### Task 6 – Product Demand Segmentation
- Feature Engineering
- K-Means Clustering
- PCA Visualization
- Inventory Recommendations

### Task 7 – Interactive Dashboard
Developed using Streamlit with:
- Sales Overview
- Forecast Explorer
- Anomaly Report
- Product Demand Segmentation

### Task 8 – Executive Business Report
Prepared a business report summarizing findings and recommendations.
